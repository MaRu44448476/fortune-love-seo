## 🎯 問題発生時の対処

### よくある問題と解決方法:

1. **データベース接続エラー**:
   - DATABASE_URLが正しく設定されているか確認
   - データベースサーバーがアクティブか確認
   - `npx prisma db push` を再実行

2. **認証エラー**:
   - Google OAuth設定のリダイレクトURL確認
   - NEXTAUTH_SECRET・NEXTAUTH_URL確認

3. **Stripe決済エラー**:
   - Stripe キーが本番/テスト環境に適合しているか確認
   - Webhook URL設定確認

4. **ページ表示エラー**:
   - Vercelの Function Logs確認
   - ブラウザ Developer Tools の Console確認
